Play Test 2

Play test 2 was pretty rough. We wanted to address a lot of issues in Play Test 1, but a lot of us were really busy and we didn't end up getting a lot of the UI features in like score keeping, user instructions, and user feedback. We did get two more games created however, and they worked flawlessly with our game cycling engine. So at least that showed us if we wanted to make 10 more games we could hook them up to our engine and the game should cycle through them without flaw.
