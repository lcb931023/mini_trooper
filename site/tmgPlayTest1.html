Play Test 1

Play test 1 was a fast and rapid production of two minigames and a game engine to cycle through our minigames. We decided to use a different framework this time around too. We are using Phaser this time around and not Pixi.js. We think Phaser offered us more and was much easier to learn. Our first playtest goals were to see if the games would cycle through each other, and if our users would understand what to do.

Goals

  • Create Nyan Cat Game
  • Create Pedobear Game
  • Game Cycling
  • Start Screen
  • Game Over Screen

Postive Results

"I like the potential of this game."
"This is real cool, I like the concept."
"The fast pace of the mini games is fun."
"Your games are have a real nice visual appeal."

Negative Results

"Running game needs some time to realize what's going on. I was caught before I hit the button most times."
"Nyan Cat game is a bit too fast."
"It's so hard! Start out easy then increase difficulty as you play through."
